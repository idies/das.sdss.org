User List Specification

Provide your list of fields as tab or comma separated value (CSV) lists. The final column, nfields, is optional. You may include comments by staring a line with a # character. Uploaded tables are limited to 10000 lines or fewer; please break up larger tables into multiple submissions. (Note that sets of consecutive fields from the same run, rerun, and camera column can be combined into a single line using the optional "number of fields" column.)

Unlike previous versions of the DAS, this version does not attempt to interpret column headings; columns must have the content and be in the order specified in the final column of "Data Release Choice" table above. Note that when requesting data from all data releases, you must specify a rerun, but not in any other cases.
